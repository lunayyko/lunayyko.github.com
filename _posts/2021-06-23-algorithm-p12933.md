## 문제 설명

함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

## 제출답안

```python
def solution(n):
    arr = list(str(n))
    arr = sorted(arr, reverse=True)
    answer = int(''.join(arr))
    print(answer)
    return answer
```

## 모범 답안

```python
def solution(n):
    ls = list(str(n))
    ls.sort(reverse = True)
    return int("".join(ls))
```

## 주요 포인트 및 생각해볼 점
sort 와 sorted, reverse 와 reversed의 차이점을 알아봐야겠다.
