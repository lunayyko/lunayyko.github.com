## 목적

레디스에 키밸류 형태로 데이터를 저장하면 메모리에 올라가서 디비에서 매번 쿼리 실행을 통해서 가져오는 것보다 더 빠르게 데이터를 가져올 수 있다.
특히 변하지 않는 값을 가져오려고 할 때 사용한다.   

### 기본 명령어

```python 
redis-cli -h 12.345.678.99 -p 6379
```
앱의 실행파일이 있는 위치에서 ip주소와 포트번호를 입력한다.

```python 
select 2 # db번호
keys * # 2번 db에 들어있는 키를 모두 보여준다

set notice 24926

```
