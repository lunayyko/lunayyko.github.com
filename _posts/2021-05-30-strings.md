# 문자열 Strings and Conditionals

``` python
def get_length(string):
    counter = 0
    for i in string:
        counter += 1
    print(counter)

get_length("Luna")
#결과
4
```

``` python
def letter_check(word, letter):
    for char in word:
        if char == letter:
            return True
    return False
```

``` python 
def contains(big_string, little_string)
    return little_string in big_string
```
편리한 'in' 을 사용하여 contain(포함)이라는 함수를 만들 수 있다.
예를 들어 "a" in "Banana" 하면 True가 반환된다.

``` python
def common_letters(string_one, string_two)
    common = []
    for letter in string_one:
    #string_one에 있는 모든 문자들 중에서 
        if (letter in string_two) and not (letter in common):
        #string_two에 있고, 이미 common에 들어있지 않은 문자라면 
            common.append(letter)
            #common에 추가한다.
    return common
```

``` python
highlighted_poems_details = [['Afterimages', 'Andre Lorde', '1997'],['The Shadow','William','1915']]
titles = []
poets = []
dates = []
for poem in highlighted_poems_details:
    title.append(poem[0])
    poets.append(poem[1])
    dates.append(poem[2])

for i in range(0, len(highlighted_poems_details)):
    print('The poem {} was published by {} in {}'.format(titles[i],poets[i],dates[i]))
```
