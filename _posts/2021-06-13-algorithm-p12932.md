## 사고 과정

자릿수를 뒤집은 다음 찢어서 배열에 넣는다.

```python
def solution(n):
    n = str(n)[::-1]
    return(list(map(int, n)))
```  

백준에서 배웠던 (list(map(int, n))) 과 a\[::-1]를 쓸 수 있어서 뿌듯했다

<br>

## 모범 답안

```python
def digit_reverse(n):
    return list(map(int, reversed(str(n))))
```
