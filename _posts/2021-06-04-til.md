# Dictionaries

* zip

``` python
numbers = [1, 2, 3]
letters = ["A", "B", "C"]
for pair in zip(numbers, letters):
    print(pair)
#결과
(1, 'A')
(2, 'B')
(3, 'C')
```

``` python
dict = {"a":1,"b":2,"c":3}
for element in dict:
    # 밸류를 프린트하려면 dict 대신 dict.value()를 사용한다
    print(element)
#결과 : 딕셔너리의 키를 프린트한다
"a"
"b"
"c"
```

## Scrabble(Dictionary 복습)
```python
letters=["A","B","C"..."y","Z"]
points = [1,3,6,... 5,10]

letters_to_point = {
    key:value
    for key, value
    in zip(letters,points)
}
letter_to_points[" "]=0

def score_word(word):
    print_total=0
    for letter in word:
        point_total += letters_to_points.get(letter,0)
    return point_total
```
