## 문제 설명

길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 내적을 return 하도록 solution 함수를 완성해주세요.

이때, a와 b의 내적은 a\[0]*b\[0] + a\[1]*b\[1] + ... + a\[n-1]*b\[n-1] 입니다. (n은 a, b의 길이)

## 제출 답안

```python
def solution(a, b):
    ans = 0
    for i in range(0,len(a)):
        ans += a[i]*b[i]
    return ans
```
한 번에 정답이 떠서 뿌듯했다.

## 모범 답안

```python
def solution(a, b):
    return sum([x*y for x, y in zip(a,b)])
```
